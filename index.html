About This Project

Pravda Network Lens

This interactive visualization was created to understand network infrastructure (domains, IP addresses, netblocks, services) of Pravda-related domains. Data was collected through passive and safe active network data collection techniques.

Data Sources
  • Active network probing
  • DNS lookups and reverse lookups
  • WHOIS data
